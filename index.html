Workflow

Three steps to ship.

Import any codebase, configure your output style, and get production-ready documentation in seconds.

1

Import your repository

Point Struct at a GitHub URL, upload a ZIP, or paste code directly. Supports monorepos, private repos via CLI, and any language.

2

Set audience & style

Choose Concise for a tight summary or Detailed for full depth. Struct adjusts structure, length, and coverage accordingly.

3

Export & integrate

Download Markdown, HTML, PDF, or OpenAPI specs. Connect to GitHub Actions for automatic updates on every push.
